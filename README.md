# Data Cleaning & Reporting Automation

## Project Description
This project automates data cleaning and reporting using the Netflix Movies and TV Shows dataset. The program handles missing values, removes duplicate records, generates dataset summaries, and creates visual reports using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Data cleaning
- Missing value handling
- Duplicate removal
- Automated reporting
- Data visualization

## Files Included
- data_cleaning.py
- netflix_titles.csv
